# Best — Curated UI/UX Skills

Hand-picked from the full library for the skills with the widest, most repeatable use in real UI/UX work — from direction and design systems through implementation, motion, review, platform guidelines, and flows.


## Design direction & taste

- **taste-skill** — Anti-slop frontend design; infers direction and ships non-templated UI. The workhorse for landing pages, portfolios, redesigns.
- **frontend-design** — Distinctive, intentional visual design for new or reshaped UI — aesthetic direction, typography, layout.
- **gpt-tasteskill** — Elite UX/UI + GSAP motion engineer; editorial typography and layout-variance discipline.
- **soft-skill** — Makes interfaces feel expensive: exact fonts, spacing, shadows, cards, animations; blocks generic patterns.
- **redesign-skill** — Audits existing sites/apps and upgrades them to premium quality without breaking functionality.
- **impeccable-design-polish** — Final-mile design polish pass for high craft.

## Design systems, color & brand

- **stitch-skill** — Generates semantic DESIGN.md design systems enforcing anti-generic UI (type, color, layout).
- **design-consultation** — Researches the product landscape and proposes a complete design system + first screens.
- **ui-ux-pro-max** — UI/UX intelligence library: 50+ styles, 161 palettes, 57 font pairings, 99 UX guidelines, chart types.
- **theme-factory** — Apply cohesive themes (colors/fonts) to artifacts, apps, slides, landing pages.
- **color-expert** — Color theory, palettes, ramps, gradients, conversions, and contrast/accessibility.
- **brand-guidelines** — Apply consistent brand colors and typography across artifacts.

## UI implementation

- **frontend-dev** — Full-stack frontend: premium UI + cinematic animation + copy + media, complete builds.
- **web-artifacts-builder** — Build elaborate multi-component web UIs with React, Tailwind, shadcn/ui.
- **image-to-code-skill** — Turn a design image into a faithful, high-fidelity website implementation.
- **swiftui-design** — Distinctive native iOS/macOS UI in SwiftUI — layout, typography, motion, anti-slop.
- **baseline-ui** — Baseline production-quality UI component conventions.

## Motion & interaction

- **gsap-core** — Core GSAP tweening, easing, stagger — the base for UI animation.
- **gsap-scrolltrigger** — Scroll-linked animation, pinning, parallax, scrub.
- **gsap-react** — useGSAP hook, refs, cleanup for React/Next UI.
- **gsap-timeline** — Sequencing and choreographing multi-step UI animations.

## Design review & QA

- **design-review** — Designer's-eye QA: spacing, hierarchy, inconsistency, AI-slop, slow interactions — then fixes.
- **fixing-accessibility** — Diagnose and fix accessibility issues in UI.
- **fixing-motion-performance** — Fix janky/expensive UI motion for smooth performance.

## Platform guidelines

- **apple-hig** — Apple Human Interface Guidelines — foundations, patterns, components, inputs (full set).
- **platform-design** — Native design conventions per platform: iOS, macOS, web, Android, watchOS, visionOS, etc.

- **hand-drawn-diagrams** — Excalidraw-style wireframes, flows, and page mockups for early UX.
