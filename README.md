# Agent Skills Collection

A curated, categorized library of **3,104 agent skills** across **12 categories** — full skills (each with its `SKILL.md` plus any assets, scripts, and references), gathered from official vendor teams and the community.

Skills are plain folders containing a `SKILL.md`. They work with Claude Code, Cowork, Codex, Cursor, Gemini CLI, and any agent that supports the standard skill format — drop a skill folder into your agent's skills directory.

Use the catalog CLI instead of copying the full repository. It validates format, exposes provenance and static-analysis signals, quarantines malformed skills, and tracks installed checksums.

```bash
python -m pip install .
skills-collection search "nextjs performance"
skills-collection inspect frontend-and-frameworks/vercel-agent-skills/vercel-optimize
skills-collection install frontend-and-frameworks/vercel-agent-skills/vercel-optimize --dest /path/to/skills
skills-collection profiles
```

> ⚠️ **Security note:** mirrored skills are unreviewed by default. Static analysis is not a security audit. Inspect a skill before giving it tool access, and read [SECURITY.md](SECURITY.md) for the trust model.
